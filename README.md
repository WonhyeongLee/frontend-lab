# Frontend-lab

## 설명

학습을 위해 새로운 환경이나 기술을 사용해서 구현한 컴포넌트들을 모아서 보기 쉽게 관리하려고 생성한 레포지토리 입니다.

## 구조

프론트관련 내용이 있는 frontend와 백엔드 관련 내용이 있는 backend로 나누고 각 각 Dockerfile을 사용해 관리합니다.

추후 docker-compose를 사용해 front와 back을 동시에 실행하도록 합니다.

[정재남님 인프런 강의](https://www.inflearn.com/course/react-vanillajs-ui%EC%9A%94%EC%86%8C%EB%A7%8C%EB%93%A4%EA%B8%B0-part1#curriculum)

구현한 UI들을 GNB에서 볼 수 있도록 구현한 부분이 제가 하려는 방법과 비슷해서 해당 부분을 참고해서 구현했습니다.
